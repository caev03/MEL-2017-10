Universidad de los Andes

Pruebas por casos

$$(p\Rightarrow r) \wedge (q\Rightarrow r) \equiv (p \vee q) \Rightarrow r$$

Pruebas por contrapositiva

$$(p \Rightarrow q) \equiv (\neg q \Rightarrow \neg p)$$

Útil cuando se puede aprovechar la doble negación para simplificar las proposiciones

Pruebas por Contradicción

$$(\neg p \Rightarrow false) \equiv (\neg false \Rightarrow \neg \neg p)$$

Cuantificadores

Una operación binaria $$\oplus$$ es ACU si: Asociativa, Conmutativa y Unitaria

  1. Suma
  2. Multiplicación
  3. & lógico
  4. | lógico

$$(\oplus x | Q : E)$$

x: Variable de cuantificación

Q: rango de la cuantificación, condición sobre x

E: cuerpo de la cuantificación

Ejercicios

  1. a max b (a,b Enteros no Negativos)
  2. a min b (a,b reales no negativos)
  3. a & b (a concatenado b)
  4. $$a \equiv b$$

Axioma Rango Vacio

$$(\oplus x | false : P ) = u$$

Ejemplos

$$(+x:int | 1\lt x \lt 0 : 5*x) = 0$$

$$(*x:int | 1\lt x \lt 0 : 5*x) = 1$$

Axioma de un punto

$$(+x:int | 0 \lt x \lt 2: 5*x)$$

$$\equiv (+x:int | x=1: 5*x)$$

$$\equiv 5*x[x:=1]$$

Axioma: Distributividad

$$(\oplus x|R:E1) \oplus (\oplus x|R:E2) = (\oplus x|R:E1\oplus E2)$$

$$(+k| 0 \lt k \lt n : k+1)$$

$$=(+k| 0 \lt k \lt n : k)+(+k| 0 \lt k \lt n : 1)$$

Axioma: Partir rango disyunto

$$(\oplus x|R\vee S:E)=(\oplus x|R:E)\oplus (\oplus x|S:E)$$

Axioma Partir rango general

$$(\oplus x|R \wedge S: E)\oplus (\oplus x|R \vee S: E)$$

$$= (\oplus x|R: E) \oplus (\oplus x|S: E)$$

Axioma: Partir rango por idempotencia

$$Si \oplus es\ \ idempotente\ \ x\oplus x = x$$

$$(\oplus x | R \vee S:E)=(\oplus x|R:E)\oplus (\oplus x|S:E)$$

Axioma Intercambio de variables de cuantificación

$$(\oplus x|R:(\oplus y|Q:E))=(\oplus y|Q:(\oplus x|R:E))$$

Axioma: Anidamiento

$$(+i,j|1\le i \le m \wedge 1\le j \le n: b[i,j])$$

$$(+i|1\le i \le m:(+y|1\le j \le n:b[i,j]))$$
